# **App Name**: BarangayConnect

## Core Features:

- User Authentication: Secure user login and signup with username, password, first name, and last name fields. First time users will be prompted with the registration page
- Profile Management: Users can manage their profiles, including personal information, contact details, and youth-related data such as civil status, work status, and youth group classifications with content validation
- Real-time Notifications: Pusher API integration for real-time event and announcement notifications. Users will get notifications in-app regarding upcoming events
- AI Chatbot (ISKAI): AI chatbot to provide FAQs and immediate responses. Display an alert with the message "Feedback sent!" when the user sends a message to the chat bot
- Accomplished Projects Carousel: Showcase accomplished projects with carousel images, replace navigation with clickable images. Each picture is displayed with a short caption.
- Interactive Event Calendar: Carousel event calendar displaying events, with event dates highlighted

## Style Guidelines:

- Primary color: Soft olive green (#A7D1AB), echoing natural growth and community connection.
- Background color: Light, desaturated pastel green (#F2FAF4), offering a calming backdrop that doesn't distract from the content.
- Accent color: Muted yellow-green (#C1DFA4) to highlight interactive elements and important information.
- Body text font: 'PT Sans' for a clean, modern look in body text. Headline font: 'Playfair' to bring an elegant feel to titles. 
- Simple, clean icons with a line art style to represent different sections and actions within the app.
- Clean and structured layout, page should scroll smoothly. Logo image (SkLogo.png) is in the upper left corner, user icon with settings is in the upper right corner
- Smooth transitions and subtle animations on user interactions.
