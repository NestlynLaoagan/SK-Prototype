# Images Folder

This folder is used to store static image assets for the application. You can place images like logos, icons, and other assets here.

Files in this `public` directory can be referenced directly from your code, starting from the root.

For example, if you have an image at `public/images/logo.png`, you can use it in a component like this:

```jsx
<img src="/images/logo.png" alt="Logo" />
```
